# Code of Conduct

## Our standard

Kern is an engineering project. Keep discussion direct, technical, and respectful.

Expected behavior:

- critique code and ideas, not people
- communicate clearly and in good faith
- assume mistakes are fixable
- keep review feedback specific and actionable
- avoid harassment, intimidation, discrimination, and personal attacks

Unacceptable behavior includes:

- abusive or hostile language
- harassment or stalking
- doxxing or publication of private information
- sustained bad-faith disruption of discussion or review
- sexualized language or imagery in project spaces

## Scope

This policy applies in project spaces, including:

- issues
- pull requests
- discussions
- code review
- other project communication channels

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, or contributions that violate this policy.

They may also temporarily or permanently restrict participation when necessary to protect the project and its contributors.

## Reporting

For serious conduct issues in this repository's GitHub spaces:

- use GitHub's private abuse/support reporting path: `https://support.github.com/contact`
- include links to the relevant issue, pull request, discussion, commit, or comment
- avoid posting sensitive details publicly

At the moment, GitHub Support is the only private reporting path published by this repository.
